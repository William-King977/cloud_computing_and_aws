# S3 using Python Boto3
## Install Boto3
1. Follow the configuration steps in the S3 `README.md`
2. Execute `pip install boto3`

## Notes
* The script (`s3_python_boto3.py`) is run inside an EC2 instance with AWSCLI already configured
